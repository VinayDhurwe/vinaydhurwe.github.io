Available Data Assets

As consultants, you'll advise on how to leverage BharatMart's existing data infrastructure:

  • Customer Transactions Data - Historical purchase patterns across channels
  • Web Interaction Data - Online browsing and engagement metrics
  • Customer Demographics & Loyalty - Segmentation and value information
  • Marketing Campaign History - Past campaign performance metrics
Consultant Note: Your recommendations should focus on how to strategically leverage these data assets within BharatMart's technical constraints.

Technical Landscape Assessment

BharatMart's operational backbone is a 1990s legacy ERP system. As consultants, you must navigate these constraints:

  • Data silos across inventory, sales, and finance modules
  • Limited or no modern APIs for programmatic access
  • High rigidity - core modifications are risky and expensive
  • Batch processing only (no real-time capabilities)
  • Running on aging on-premise infrastructure
Strategic Insight: The real challenge isn't just the AI algorithm - it's integrating with this legacy system without disrupting core operations.

Technical Integration Strategy

Recommend an approach for integrating with BharatMart's legacy ERP system:

A
API Layer Development

Digital Wrapper Strategy: Build modern middleware layer as intermediary. Exposes controlled ERP functionalities via APIs without core modifications.

  • ✓ Minimizes ERP disruption
  • ✓ Future-proof integration
  • ⚠ Potential data latency
B
Data Lake/Warehouse

Centralized Cloud Repository: Extract data from legacy systems into cloud data lake. AI models access unified data without touching ERP.

  • ✓ Single source of truth
  • ✓ Decouples AI from ERP
  • ⚠ Significant upfront investment
C
Incremental Microservices

Gradual Modernization: Carve out specific ERP functionalities into modern microservices (e.g., Customer Profile Service).

  • ✓ Modernizes critical components
  • ✓ Improves agility
  • ⚠ High architectural complexity
Be specific about what will be delivered in each phase, how you'll measure success at each stage, and how you'll address legacy system integration challenges.
